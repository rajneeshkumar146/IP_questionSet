Input Format
--------------------------------------------------------------------------------
N (integer) - number of vertices
E (integer) - number of edges
E lines of triplets of space separated integers representing start vertex, end
vertex and weight of E edges.
srcVertex (integer) minWeight (integer)
--------------------------------------------------------------------------------
Constraints
--------------------------------------------------------------------------------
1 <= N <= 10
1 <= E <= NC2
0 <= Vi1, Vi2 <= N - 1
0 < weight <= 10^9
0 <= srcVertex <= N - 1
0 < minWeight < 10^9
--------------------------------------------------------------------------------
