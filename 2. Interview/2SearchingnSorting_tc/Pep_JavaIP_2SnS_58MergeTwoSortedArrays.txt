--------------------------------------------------------------------------------
Input Format
--------------------------------------------------------------------------------
N1 (integer) - number of elements in first array
N1 space-separated integers - contents of first array
N2 (integer) - number of elements in second array
N2 space-separated integers - contents of second array
--------------------------------------------------------------------------------
Constraints
--------------------------------------------------------------------------------
1 <= N1, N2 <= 10^5
-10^9 <= Ai, Bi <= 10^9, where Ai and Bi are elements of first and second array
--------------------------------------------------------------------------------
Output Format
--------------------------------------------------------------------------------
1. Print on first line space-separated integers of first array, after merge.
2. Print on second line space-separated integers of second array, after merge.
