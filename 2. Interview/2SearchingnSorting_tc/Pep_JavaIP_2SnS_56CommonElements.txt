Input Format
--------------------------------------------------------------------------------
N1 (integer) - number of elements of first array
N1 space-separated integers - contents of first array
N2 (integer) - number of elements of second array
N2 space-separated integers - contents of second array
N3 (integer) - number of elements of third array
N4 space-separated integers - contents of third array
--------------------------------------------------------------------------------
Constraints
--------------------------------------------------------------------------------
1 <= N1, N2, N3 <= 10^5
-10^9 <= Ai, Bi, Ci <= 10^9 where Ai, Bi and Ci are elements of first, second
and third array
--------------------------------------------------------------------------------
Output Format
--------------------------------------------------------------------------------
1. Print all common elements on the same line in a space separated manner.
