Input Format
--------------------------------------------------------------------------------
N (integer) - size of array
N spaced pairs of integers (A1, B1)...(Ai, Bi)...(AN,BN), where Ai is the start
point of ith interval and Bi is the end point of the ith interval. 
--------------------------------------------------------------------------------
Constraints
--------------------------------------------------------------------------------
0 <= N <= 10^5
-10^9 <= Ai, Bi <= 10^9
--------------------------------------------------------------------------------
Output Format
--------------------------------------------------------------------------------
Print the merged pairs of interval in sorted order.
