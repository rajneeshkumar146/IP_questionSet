--------------------------------------------------------------------------------
Input Format
--------------------------------------------------------------------------------
N (integer) - size of N * N matrix
N * N integers - elements of array (Aij - assigned row-wise)
--------------------------------------------------------------------------------
Constraints
--------------------------------------------------------------------------------
0 <= N <= 10^5
-10^9 <= Aij <= 10^9
--------------------------------------------------------------------------------
Output Format
--------------------------------------------------------------------------------
1. Print the clockwise rotation row-by-row, each row in next line and elements
of a row separated by space.
2. Print the anti-clockwise rotation row-by-row, each row in next line and
elements of a row separated by space.
